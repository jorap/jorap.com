**Markdown** is lightweight markup - `#` for headings, `*` for emphasis, links in brackets. Hugo, GitHub, TiddlyWiki (with plugins), Obsidian - they all speak variants.

Learning Markdown once paid off everywhere I publish or capture.

---

## Core syntax I live in

Headings, lists, links, images, code fences, blockquotes. **90% of posts use only these.**

---

## Frontmatter is separate

YAML between `---` at top - Hugo metadata. **Not strict Markdown**, but always paired with it in SSG world.

---

## Flavors differ

GitHub tables aren't everywhere. Obsidian wikilinks `[[ ]]` aren't standard. **Know your target renderer.**

---

## Bottom line

Markdown deep dive worth it means mastering the boring basics and code blocks. **Plain, portable, future-proof.** Fancy extensions are optional seasoning.
